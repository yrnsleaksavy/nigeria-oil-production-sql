This project consists of the compilations of the Nigeria Oil and Gas production output for the part year of 2025.

-------------------------------------------

The raw data set is from the uploaded excel spreadsheet on the NUPRC site.

------------------------------------------
Few Questions Answered with SQL:

1. Which month had the highest oil production in the past months

2. Highest production across terminal

3. Highest production volume overall

4. Which terminal (or region) contributes the most to export volumes?
